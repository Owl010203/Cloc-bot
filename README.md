# 🤖 Бот учёта рабочего времени

Telegram-бот для отметки прихода/ухода сотрудников.  
Генерирует JSON-файл, совместимый с HTML-приложением «Учёт зарплат».

---

## 🚀 Быстрый старт (Railway — бесплатно, без своего сервера)

### Шаг 1 — Создай бота в Telegram
1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Напиши `/newbot`
3. Придумай имя и username боту
4. Скопируй **токен** (выглядит как `1234567890:AAF...`)

### Шаг 2 — Узнай свой Telegram ID (для прав администратора)
1. Напиши [@userinfobot](https://t.me/userinfobot) — он пришлёт твой ID
2. Запомни число (например: `123456789`)

### Шаг 3 — Деплой на Railway
1. Зарегистрируйся на [railway.app](https://railway.app) (можно через GitHub)
2. Нажми **New Project → Deploy from GitHub repo**
3. Загрузи файлы из этой папки в GitHub репозиторий (или нажми **Deploy from local** через Railway CLI)
4. В настройках проекта Railway → **Variables** добавь:
   ```
   BOT_TOKEN = токен_от_BotFather
   ADMIN_ID  = твой_telegram_id
   ```
5. Railway автоматически запустит бота!

## 🖥️ Запуск локально (на своём компьютере)

```bash
# Установи Python 3.10+
pip install -r requirements.txt

# Windows (PowerShell):
$env:BOT_TOKEN="твой_токен"
$env:ADMIN_ID="твой_id"
python bot.py

# macOS/Linux:
BOT_TOKEN="твой_токен" ADMIN_ID="твой_id" python bot.py
```

## 📖 Как пользоваться

### Сотрудники:
1. Найти бота в Telegram по username
2. Написать `/start` — ввести своё имя (как в списке)
3. Нажимать **🟢 Пришёл** и **🔴 Ушёл** каждый день
4. Смотреть свою статистику через **📊 Мои часы**

### Администратор:
1. Написать `/start` — получишь кнопку **👑 Админ**
2. Через **👑 Админ** можно:
   - 📅 Создать новый период
   - 👥 Управлять сотрудниками
   - 📋 Смотреть статистику текущего периода
   - 📥 **Скачать JSON** — готовый файл для HTML-приложения
3. Или написать `/export` — бот сразу пришлёт файл

### Загрузка в HTML-приложение:
1. В боте нажми **📥 Скачать JSON**
2. В HTML-приложении нажми **📂 Загрузить данные**
3. Выбери скачанный файл — всё загрузится!

## 💡 Советы

- Бот хранит данные в `data.json` — не удаляй этот файл!
- На Railway данные сохраняются между перезапусками (используй Railway Volume для надёжности)
- Если сотрудник ошибся со временем — ты как админ можешь исправить напрямую в HTML-приложении, потом снова выгрузить JSON и обновить файл в боте через Railway
